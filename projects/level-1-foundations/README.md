# Level 1 — Foundations (Week 1-4)

Goal: get fluent with Python + Git, build first analysis tool for sensor data.

No hardware needed. Pure laptop work.

## Days

Each day = 1 folder with `README.md` (the task) + `main.py` (starter code).

Folders auto-generated each morning by `automation/daily_template.py` (or run it manually).

## After 28 days you have

- One polished GitHub repo (this one) demonstrating:
  - CSV / data analysis
  - Anomaly detection logic
  - CLI tool design
  - OOP basics
  - pytest tests
  - GitHub Actions CI
  - Recruiter-ready README

Then you move on to Level 2 — Industrial Protocols (Modbus / CAN / MQTT).

## Sample data

`shared/` contains shared sample CSV files used across multiple days, so each day can focus on the new concept without re-generating data.
